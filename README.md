# Umibots landing page

A single-screen prospect page for Clodo Gonzalez and the Umibots team, built in their own aqua-on-black industrial vocabulary. The page carries the fit-analysis pitch body verbatim (headline, 60/30/10 frame, Feeld case, ICM paper, community credibility line, CTA to Matt Creamer's Calendly), strips all Eduba engagement pricing, hyperlinks every URL with target blank and rel noopener, and keeps Eduba off the page except for the footer attribution and the named case references. Flat geometry and zero corner radius mirror the Umibots site. Fonts are Inter only (a web-safe fallback for the proprietary Gilmer face Umibots uses). Renders cleanly at 375px.
